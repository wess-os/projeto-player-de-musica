<h1 align="center"> Projeto Player De música </h1>

<h3>🧾 Descrição</h3>
Esse projeto reproduz música no navegador de forma exemplar, tem opção para reduzir volume, avançar, pausar e voltar as músicas

<h3>💻 Tecnologias utilizadas:</h3>
HTML, CSS and JS
